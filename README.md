# wsh-cli
![Npm 版本](https://img.shields.io/badge/wsh-cli_0.0.1-pink)
## 📖简介
  - 用于快速搭建前端项目的命令行工具
  
## 🎉 模板
  - 🔍 快速生成[Vue3+TS+Vite+ElementPlus+Pinia+Axios+Mock的模板](https://gitee.com/wu-shihao05/custom-scaffold.git)
## ✨ 安装
  ```bash
  npm install wsh-cli -g
  ```
## 💎 使用
```bash
# 使用模板创建新项目
wsh create

# 执行项目文件夹名称并选择模板创建新项目

wsh create [projectName]

# 查看脚手架版本
wsh -v
wsh --version
```
